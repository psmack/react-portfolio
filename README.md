# Portfolio Website

This portfolio website is built with Tailwind CSS and Next.js framework, a React based web application.

## Getting Started

Run the development server:

```
npm run dev
or
yarn dev
or
pnpm dev
```
Build for production:

```
npm run build
npm run start
```
